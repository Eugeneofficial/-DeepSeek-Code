# DeepSeek Code 2025

üöÄ **DeepSeek Code Companion** is an AI-powered coding assistant that runs locally on your machine. Built with Gradio, LangChain, and powered by the DeepSeek-r1 language model via Ollama, it offers intelligent coding assistance, debugging help, and model management. Assembled by Eugeneofficial.

## Features
- üêç **Python Expertise**: Code completion, bug fixing, best practices, and performance tips.
- üêû **Debugging Assistant**: Strategic print statements, error analysis, step-by-step problem solving.
- üìù **Code Documentation**: Automatic docstrings, explanations, and comment suggestions.
- üí° **Solution Design**: Architecture recommendations, design patterns, and scalability considerations.
- üåê **Model Management**: Download and manage models from the [Ollama Library](https://ollama.com/library).

## Installation
- **Requirements**: Only `setup.bat` is needed.
- **Steps**:
  1. Download or clone this repository.
  2. Place `setup.bat` in the root directory.
  3. Double-click `setup.bat` and follow the prompts to set the model directory.
  4. Wait for the installation to complete and open `http://127.0.0.1:7860` or the shared URL.

## Usage
- **Chat**: Type your coding question and select a model (1.5b or 3b).
- **Download Models**: Enter commands like `ollama pull deepseek-r1:1.5b` in the "Download Model" field.
- **Stop Downloads**: Click "Stop Download" to cancel.
